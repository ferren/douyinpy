抖音开发平台 Python SDK。（参照 wechatpy 的实现形式）


## 功能特性

1. 抖音开放平台 API
2. 抖音小程序 API

## 安装

推荐使用 pip 进行安装:

```bash
pip install douyinpy
```

升级版本：

    pip install -U douyinpy
